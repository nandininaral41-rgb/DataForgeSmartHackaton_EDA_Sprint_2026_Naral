# Practical 2 Report – Excel Data Cleaning and Quality Checks

## Objective
The objective of this practical was to clean the dataset in Microsoft Excel and perform quality checks to ensure the data is accurate, consistent, and ready for analysis.

## Dataset Used
- **Dataset Name:** `dataset_raw.csv`
- **Source:** Project dataset
- **Tool Used:** Microsoft Excel

## Procedure
1. Opened the dataset in Microsoft Excel.
2. Checked for missing (blank) values.
3. Identified and removed duplicate records.
4. Corrected inconsistent data entries where required.
5. Verified the data types and formatting of each column.
6. Ensured there were no unnecessary spaces or invalid values.
7. Saved the cleaned dataset for further analysis.

## Quality Checks Performed
- ✔ Checked for missing values.
- ✔ Checked for duplicate records.
- ✔ Verified column names.
- ✔ Ensured consistent formatting.
- ✔ Validated numeric and text fields.
- ✔ Confirmed the dataset was ready for analysis.

## Observations
- No duplicate records were found.
- No missing values were present in the dataset.
- The dataset was consistent and properly formatted.
- Data quality was satisfactory for further processing.

## Outcome
The dataset was successfully cleaned and validated using Microsoft Excel. It is now ready for exploratory data analysis and visualization.

## Conclusion
This practical provided hands-on experience in cleaning and validating data using Microsoft Excel. Performing data quality checks improves the accuracy and reliability of future analysis.
