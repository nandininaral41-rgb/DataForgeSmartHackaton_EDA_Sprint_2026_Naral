# Practical 9: Statistical EDA & Correlation Analysis

## Objective
To perform exploratory data analysis using descriptive statistics, grouped tables, ranking, and correlation analysis to identify patterns and relationships in the dataset.

## Work Completed
- Generated a numeric summary using `describe()`.
- Exported the numeric summary as `numeric_summary.csv`.
- Created and exported the missing value percentage table.
- Created four grouped tables using `groupby()`.
- Created Top 10 and Bottom 10 ranking tables based on Sales.
- Generated the correlation matrix and exported it as `correlation_matrix.csv`.
- Calculated the skewness of the Sales column.
