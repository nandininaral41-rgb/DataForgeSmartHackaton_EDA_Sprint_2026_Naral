# Practical 3: Excel/Google Sheets EDA Dashboard

## 1. Which pivot table gave the most useful insight?

The Pivot Table based on **Region/State/City** gave the most useful insight. It helped identify which geographical areas had higher or lower sales performance. This supports better decision-making for targeting and improving underperforming regions.

---

## 2. What changed when you used slicers?

When slicers were used, the dashboard became interactive and dynamic. Instead of manually filtering data, slicers allowed real-time filtering of all connected pivot tables and charts. This made analysis faster, easier, and more user-friendly.

---

## 3. What is one limitation of doing this only in Excel/Google Sheets?

A major limitation of Excel/Google Sheets is that it cannot efficiently handle very large datasets or advanced analytics. It also lacks automation and scalability compared to tools like Python, Power BI, or databases.

---

## 4. What is KPI?

**KPI (Key Performance Indicator)** is a measurable value that shows how effectively a business or system is performing against its goals.

In this dashboard, KPIs represent important summary metrics such as:
- Total Sales
- Average Sales
- Total Number of Records

KPIs help users quickly understand overall performance without analyzing raw data.
