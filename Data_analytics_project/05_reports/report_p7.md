# Practical 7: Python DataFrame Setup and Schema Validation

## Objective
To load a CSV dataset using Pandas, inspect its structure, validate the schema, and export the schema audit table.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Steps Performed
1. Imported the Pandas library.
2. Loaded the CSV dataset into a DataFrame.
3. Displayed the dataset shape.
4. Viewed the first and last five rows.
5. Displayed column names and data types.
6. Generated summary statistics.
7. Checked missing values.
8. Created a schema audit table containing:
   - Column Name
   - Data Type
   - Missing Count
   - Unique Count
   - Sample Value
9. Exported the schema audit table as `schema_audit.csv`.

## Conclusion
The dataset was successfully loaded and inspected. Schema validation helped identify the structure, missing values, and data types of each column. The generated schema audit improves data quality assessment before further analysis.
