## 🌐 Auto EDA Web Application

A lightweight browser-based application that performs automatic data analysis from uploaded CSV files.

### Features

- Upload CSV file
- Display dataset shape (rows and columns)
- Show column names
- Preview dataset
- Simple and interactive UI
- Runs directly in browser without backend
